# Arquitetura do Sistema

## Fluxo principal
Cliente → Empreendimento → Unidade → Venda → Contrato → Financeiro

## Fluxos auxiliares
Empreendimento → Memorial descritivo
Empreendimento → Planta
Empreendimento → Tabela de venda
Empreendimento → Espelho de vendas

## Módulos do sistema
1. Dashboard
2. Empreendimentos
3. Espelho de vendas
4. Clientes
5. Contratos
6. Financeiro

## Fonte inicial dos dados
- Mock local em arquivos JSON/JS
- Links externos para Drive/Formulários
- Integração posterior com n8n
