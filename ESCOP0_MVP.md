# Escopo do MVP — Sistema Interno De Marchi

## Objetivo
Criar um sistema interno simples para centralizar informações operacionais e financeiras dos empreendimentos, integrando clientes, espelho de vendas, documentos e automação de contratos.

## Entram no MVP
- Dashboard
- Empreendimentos
- Espelho de vendas
- Clientes
- Contratos
- Financeiro básico
- Links para memorial, plantas e tabela de vendas

## Ficam para depois
- Login real
- Integração completa com ERP
- Histórico automático completo
- Permissões por usuário
- Assinatura digital integrada
- Integrações em tempo real com n8n

## Observações
- Os Forms já existem
- O n8n já está pensado
- O sistema visual será uma camada operacional simples
- Inicialmente haverá dados mockados
